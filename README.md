# Finalware

## Introduction
A simple walktrough Firmware Updater for your Starlight-12 mouse. Cross-Platform, User Friendly and built on electron.

## Support 
We currently support M1 Macs, Windows and Linux.

## Installation
### Windows
On Windows we got a msi oneclick installer, its the most user friendly experience out of the 3 supported platforms.
### Linux
We provide a .deb package for debian based system aswell as a zip file with a binary file. Besides that you currently have to change ur user permissions to access 
COM Ports.
